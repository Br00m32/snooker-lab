# 🎱 SnookerLab

Приложение для учёта тренировок по снукеру. Работает как PWA — можно установить на телефон.

## 🚀 Деплой на GitHub Pages (бесплатно)

### Шаг 1: Создай репозиторий
1. Зайди на [github.com/new](https://github.com/new)
2. Назови репозиторий `snooker-lab`
3. Оставь **Public**, нажми **Create repository**

### Шаг 2: Загрузи код
```bash
cd snooker-lab
git init
git add .
git commit -m "initial commit"
git branch -M main
git remote add origin https://github.com/ТВОЙ_ЮЗЕРНЕЙМ/snooker-lab.git
git push -u origin main
```

### Шаг 3: Включи GitHub Pages
1. Зайди в **Settings** → **Pages** (в меню слева)
2. В разделе **Source** выбери **GitHub Actions**
3. Всё! После пуша автоматически задеплоится

### Шаг 4: Открой на телефоне
1. Перейди на `https://ТВОЙ_ЮЗЕРНЕЙМ.github.io/snooker-lab/`
2. Нажми **«Поделиться»** → **«На экран Домой»** (iOS)
3. Или **меню ⋮** → **«Добавить на главный экран»** (Android)
4. Готово! Работает офлайн как нативное приложение 🎯

## 💻 Локальная разработка

```bash
npm install
npm run dev
```

Откроется на `http://localhost:5173/snooker-lab/`

## ⚠️ Важно

Если твой юзернейм на GitHub **не** совпадает с названием репозитория `snooker-lab`, всё будет работать автоматически. Но если ты хочешь переименовать репо, нужно также поменять `base` в `vite.config.js`.

## 📱 Возможности

- Создание тренировок с автоматическим названием (дата + время)
- 3 упражнения: Line-up, The T, Straight Blue
- Чередование красный → цветной (с исключением для Straight Blue)
- Статистика: лучший/средний брейк, % забивания, Red→Color %, стабильность
- График прогресса по тренировкам
- Тёмная тема, крупные кнопки — удобно у стола
- PWA: работает офлайн, устанавливается на телефон
- Данные хранятся локально в браузере
